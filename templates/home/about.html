About Click-Bait Haven

Hello and welcome to my little web app!

This is supposed to be a journalism-style website in which you can submit articles for review, tag articles, comment, like or dislike articles and manage your personal profile.

The article content of this site support the MarkDown Markup Language but keep in mind that the available tags are:

'p', 'b', 'strong', 'em', 'ul', 'ol', 'li', 'h1', 'h2', 'h3', 'h4', 'h5', 'h6', 'pre', 'code'

I also built my own REST API which allows for anyone to dynamically fetch articles programatically! No key needed!

The main idea for making this project was seeing if I could develop a realistic web application with user privileges, moderation and custom logic.

Its a culmination of techniques I learned through lectures, studies, and my own experimentation.

Since im also hobby-watching a lot of scripting and cybersecurity content I made sure to include a lot of checks in my code and custom scripts for various purposes.
